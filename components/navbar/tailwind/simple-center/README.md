# Simple Center Navbar

A clean, responsive navigation bar with centered logo and menu items.

## Features

- Responsive design with mobile hamburger menu
- Clean and minimal styling
- Smooth hover transitions
- Mobile-first approach

## Usage

Copy the HTML code from `index.html` and include Tailwind CSS in your project.

## Customization

- Modify the logo text or replace with an image
- Update navigation links in the menu
- Customize colors by changing Tailwind classes
- Add additional menu items as needed

## Browser Support

Works in all modern browsers that support CSS Grid and Flexbox.
